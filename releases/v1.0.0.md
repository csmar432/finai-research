# Release Notes — v1.0.0 (2026-06-15)

> **Milestone release** · 经济金融领域 AI 学术研究工作流首个 GA 版本。

我们很高兴发布 **FinAI Research Workflow v1.0.0**——一个面向经济金融研究者的端到端 AI 工作流。
从研究想法到可投稿论文，全流程自动化，集成 43 个 MCP 数据服务器、42 种计量方法、17 个 AI 技能和 6 大中文顶刊投稿标准。

---

## ✨ 核心亮点

### 📊 数据生态：43 MCP Servers
- **学术文献** (5)：OpenAlex 2亿+ 论文 · ArXiv · Context7 全文 · Semantic Scholar · NBER
- **A股数据** (4)：Tushare · 东方财富 · Wind · CSMAR
- **美股/全球** (3)：Yahoo Finance · SEC EDGAR · 增强金融
- **宏观经济** (10)：中国/全球宏观 · 美联储 · 世界银行 · IMF · OECD · BEA · CEIC
- **研报/新闻/工具/加密** (27)：东方财富研报、NewsAPI、CryptoCompare、LaTeX MCP、Playwright、E2B 等

### 🧮 42 种计量方法（JF/JFE/RFS 标准）
- **DID 家族** (12)：Callaway-Sant'Anna · Sun-Abraham · Borusyak · Goodman-Bacon · dCdH · 合成 DID · 三重差分
- **IV/GMM/空间** (7)：IV/2SLS · Panel GMM · Arellano-Bond · Blundell-Bond · 交互固定效应 · 门槛回归 · 空间 SDM/SAR/SEM
- **时间序列** (8)：GARCH/GJR/EGARCH · DCC-GARCH · TVP-VAR · Panel VAR · 面板协整
- **离散/生存/因果 ML** (12)：Logit/Probit · Cox PH · Causal Forest · DoubleML · T-Learner/X-Learner
- **数据/图表/输出** (10)：data_fetcher · fin_charts · report_generator · a_share_variables · policy_database

### 🤖 17 个 AI 技能
- **发现阶段**：fin-idea-discovery · fin-generate-idea · fin-novelty-check · fin-lit-review
- **设计阶段**：fin-experiment-design · fin-data-acquisition
- **写作阶段**：fin-paper-plan · fin-paper-draft · fin-paper-figure · fin-paper-writing · fin-paper-convert
- **发布阶段**：fin-review-loop · fin-submit-check · fin-ref-paper · fin-brief-generator · fin-viz-launch

### 📄 论文写作（45 种期刊模板）
- **英文顶刊**：JF · JFE · RFS · JAE · JFQA · RSTUD · QJE · Econometrica
- **中文顶刊**：经济研究 · 金融研究 · 管理世界 · 会计研究 · 中国工业经济
- LaTeX 编译 · 图表 ≥300 DPI · BibTeX 引用 · PRISMA 合规

### 🛡️ 工程化与可复现性
- 84 个测试文件 · 7 个 CI jobs · 3 平台 matrix (Ubuntu/macOS/Windows) · coverage 报告
- 4 层数据 fallback：MCP → Python lib → HTTP → synthetic（带显式标记）
- 强制交互 Checkpoint (HITL) · 数据溯源追踪 (provenance)
- 沙箱 AST 验证 · Pre-commit hooks · Dependabot 自动更新

## 🚀 Quick Start

```bash
# 1. 安装
git clone https://github.com/csmar432/finai-research-workflow.git
cd finai-research-workflow
pip install -r requirements.txt

# 2. 配置 API Key
cp .env.example .env
# 编辑 .env，至少填 DEEPSEEK_API_KEY

# 3. 健康检查
python scripts/health_check.py

# 4. 启动研究
python scripts/agent_pipeline.py --topic "碳排放权交易对企业绿色创新的影响"
```
