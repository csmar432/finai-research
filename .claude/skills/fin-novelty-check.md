# fin-novelty-check — 新颖性验证

验证经济金融研究想法的新颖性。在 JF/JFE/RFS/JME 等顶刊及 arXiv 中搜索近三年文献，输出结构化新颖性报告和定位策略。

## 功能

### 想法解析

从研究想法中提取 3-5 个核心主张：
- 研究问题
- 研究方法
- 数据/样本情境
- 机制解释
- 创新维度

### 多源文献搜索

| 数据库 | 覆盖 |
|--------|------|
| NBER Working Papers | 美国宏观/公司金融 |
| 中文顶刊（WebSearch）| 经济研究/金融研究/管理世界 |
| 研报（EastMoney）| 补充来源 |

### 新颖性评估

**评分维度**：
- 研究问题相似度（1-10）
- 方法创新度（1-10）
- 样本独特性（1-10）
- 机制解释（1-10）

**综合评分**：加权平均 → X/10

**评级**：
- HIGH（8+）：显著边际贡献，定位清晰
- MEDIUM（6-8）：有一定边际贡献，需策略调整
- LOW（<6）：与现有研究高度重叠，建议换角度

### 输出

`output/fin-novelty/NOVELTY_REPORT.md` — 包含：
- 新颖性评分（X/10）
- 评级（HIGH/MEDIUM/LOW）
- 定位策略
- Reviewer 风险预警（哪些点会被质疑）
- 3 篇最相关参考文献

## 调用方式

```
"验证一下这个想法的新颖性：关税政策对A股出口型企业创新的影响"
```
