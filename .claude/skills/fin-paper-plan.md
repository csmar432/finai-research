# fin-paper-plan — 论文大纲生成

为经济金融论文生成结构化大纲。根据 `FIN_BRIEF.md` 中的目标期刊，适配相应的格式要求、字数限制和结构规范。

## 功能

### 论文类型模板

| 类型 | 说明 |
|------|------|
| 实证研究 | DID/IV/RD 方法，含假设推导、机制分析 |
| 文献综述 | 系统梳理研究脉络，识别缺口 |
| 方法创新 | 提出新计量方法/因子/算法 |

### 期刊适配

**英文顶刊**：JF / JFE / RFS / JAE / AER / Econometrics Journal

**中文顶刊**：
- 经济研究 — 假说驱动、理论推导严密、稳健性充分
- 金融研究 — 实证导向、金融市场特征突出
- 管理世界 — 政策含义丰富、管理启示明确

### 输出文件

| 文件 | 说明 |
|------|------|
| `PAPER_OUTLINE.md` | 完整章节结构 |
| `FIGURE_PLAN.md` | 图表计划（含位置、类型、数据要求）|
| `TABLE_PLAN.md` | 表格计划（含位置、列、格式）|
| `ABSTRACT_DRAFT.md` | 摘要草稿模板 |

## 调用方式

```
"生成一篇发表在经济研究的论文大纲，研究方向是企业ESG"
```
