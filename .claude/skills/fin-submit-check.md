# fin-submit-check — 投稿前检查

根据 `PAPER_OUTLINE.md` 中的目标期刊要求，逐一检查论文的格式、图表、引用、参考文献、数据可用性等所有投稿必需项。

## 功能

### 检查项（10 大类）

| 检查项 | 说明 |
|--------|------|
| 匿名检查 | 双盲审稿匿名性（作者信息、致谢、资助信息）|
| 字数/格式 | 目标期刊字数限制、格式规范 |
| 图表分辨率 | ≥300 DPI，EPS/PDF 格式 |
| 引用完整性 | 无空白引用、无未核实引用 |
| 参考文献格式 | JF/JFE/RFS/GB-T-7714 等 |
| 公式编号 | 全文编号连续且无遗漏 |
| 数据可用性 | 数据集说明、代码可用性 |
| 附录检查 | 互联网附录完整性 |
| 语法校对 | Academic Proofreading |
| LaTeX 编译 | 无编译错误、无警告 |

### PDF 自动化验证

- 图表 DPI 检查（Python 脚本）
- 引用审计（正则表达式）
- 编译错误扫描

## 输出

| 文件 | 说明 |
|------|------|
| `SUBMIT_CHECK_REPORT.md` | 检查报告 |
| `SUBMIT_CHECK_ISSUES.md` | 问题清单 |

## 调用方式

```
"检查一下我的论文是否满足经济研究的投稿要求"
```
