# fin-review-loop — 对抗性 Review 循环

对草稿进行多轮严格评审，检查实证严谨性、方法正确性、理论贡献和写作质量，直到达到发表标准。

## 功能

### 评审维度

| 维度 | 权重 | 评分范围 |
|------|------|---------|
| 新颖性（Novelty）| 30% | 1-10 |
| 实证严谨性（Rigor）| 30% | 1-10 |
| 文献覆盖（Literature）| 15% | 1-10 |
| 写作清晰度（Clarity）| 15% | 1-10 |
| 学术影响（Impact）| 10% | 1-10 |

### 通过阈值

- 所有维度 ≥ 最低分
- 加权平均 ≥ 7.5
- 最多 4 轮

### 阻断条件

- 新颖性 < 6.0 → 立即停止，建议重做
- 实证严谨性 < 6.0 → 立即停止，必须修复

### Review 类型

| 类型 | 说明 |
|------|------|
| Standard | 标准学术评审 |
| Strict | 更严格的评判标准 |
| Nightmare | 模拟顶刊拒稿理由 |

### 评审模板

- 实证论文模板（DID/IV 重点）
- 中文期刊模板（假说推导、机制检验、政策启示）

## 输出

`output/fin-review/REVIEW_REPORT.md` — 包含：
- 各维度评分
- 具体问题列表
- 修改建议
- Reviewer 视角风险评估

## 调用方式

```
"review一下我的关税政策论文草稿"
```
