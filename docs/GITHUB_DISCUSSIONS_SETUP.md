# GitHub Discussions Setup Guide

GitHub Discussions must be enabled manually (no API available).

## How to Enable

1. Go to your repo: https://github.com/csmar432/FinAI-Research-Workflow
2. Click **Settings** (top navigation bar)
3. Scroll down to **Features** section
4. Toggle **Discussions** ON
5. Configure categories (see below)

## Recommended Categories

| Category Name | Description | Template |
|---|---|---|
| Ideas | Feature requests and research direction proposals | [Blank] |
| Q&A | Questions about usage, installation, and methods | Enable Q&A |
| General | Open discussion, announcements, show-and-tell | [Blank] |
| Announcements | Project updates, releases, major changes | Moderators only |

## Why Discussions?

- **Lower friction** than Issues for questions
- **Searchable** Q&A that helps future users
- **Community building** without maintaining a separate forum
- **Issue tracker** stays clean for bugs and actionable tasks
