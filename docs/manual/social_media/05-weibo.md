## 提交步骤

1. 打开 https://weibo.com
2. 登录微博账号
3. 点击右上角 "发微博" 按钮
4. 粘贴正文
5. 上传 1-9 张图
6. 添加话题：
   - #计量经济学#
   - #DID双重差分#
   - #开源工具#
   - #金融研究#
   - #经济研究#
   - #学术写作#
7. 点击 "发送"
