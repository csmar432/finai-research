# 论文写作工作流使用指南 v2.0

## 核心原则

1. **先大纲，后内容** - 不会直接生成完整论文
2. **数据必须真实** - 禁止编造任何数据
3. **用户确认每步** - 每章节完成后询问用户

## 工作流程

```
┌─────────────────────────────────────────────────────────────┐
│  步骤1: 确定研究主题                                        │
│  ↓ 用户描述研究方向                                        │
│  ↓ 系统生成备选题目                                        │
│  ↓ 用户确认/选择题目                                       │
├─────────────────────────────────────────────────────────────┤
│  步骤2: 生成论文大纲                                        │
│  ↓ 系统生成完整大纲                                        │
│  ↓ 用户确认/修改大纲                                        │
├─────────────────────────────────────────────────────────────┤
│  步骤3: 逐章节写作                                          │
│  ↓ 系统生成第1章                                           │
│  ↓ 用户确认/修改                                          │
│  ↓ 系统生成第2章                                           │
│  ↓ 用户确认/修改                                          │
│  ↓ ... 直至完成所有章节                                    │
├─────────────────────────────────────────────────────────────┤
│  步骤4: 数据分析与实证（可选）                              │
│  ↓ 用户选择是否生成真实数据                                │
│  ↓ 如需数据，调用 econometrics.py                          │
├─────────────────────────────────────────────────────────────┤
│  步骤5: 整合输出                                            │
│  ↓ 合并所有章节                                            │
│  ↓ 生成完整论文草稿                                        │
└─────────────────────────────────────────────────────────────┘
```

## 重要说明

### 关于数据

**本工作流不会自动编造数据！**

如果你需要真实的实证分析结果：
1. 准备好数据文件（CSV/Excel）
2. 运行 `python scripts/econometrics.py` 进行分析
3. 将结果填入论文相应位置

### 当前可用的模块

| 模块 | 功能 | 状态 |
|------|------|------|
| `interactive_paper_pipeline.py` | 交互式论文写作 | ✅ 可用 |
| `econometrics.py` | 计量回归分析 | ✅ 可用 |
| `data_pipeline.py` | 数据清洗 | ✅ 可用 |
| `paper_full_pipeline.py` | 完整流程（需真实数据） | ⚠️ 需数据 |
| `dashboard.py` | 可视化界面 | ✅ 可用 |

## 注意事项

1. **没有LLM API也能使用** - 可以生成大纲结构
2. **数据必须自备** - 实证分析需要你自己的数据
3. **随时可中断** - 按 Ctrl+C 保存进度
4. **草稿自动保存** - 中断后可继续

## 常见问题

### Q: 为什么生成的论文没有数据？
A: 这是正确的设计！本工作流先生成论文框架，数据需要另行分析后填入。

### Q: 如何进行实证分析？
A: 运行 `python scripts/econometrics.py`，准备好CSV数据文件即可。

### Q: 可以导出Word吗？
A: 可以，运行 `python scripts/generate_docx_tables.py`
