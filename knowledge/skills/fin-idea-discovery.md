# fin-idea-discovery — 经济金融研究想法发现流程

> **操作版本**：`.cursor/skills/fin-idea-discovery/SKILL.md`
> **知识源版本**：本文件

## 功能描述

从研究方向出发，经过系统化的想法发现、数据验证和细化，输出经过实证验证的可执行研究方案。

## 工作流

```
研究方向 → 文献综述 → 想法生成(8-12个) → 数据可行性验证 → 用户确认 → 输出候选想法
```

## 新增原则（2026-06-04）

### 想法-数据交叉验证

在阶段 2 想法生成后、阶段 3 新颖性验证前，**必须**展示数据可行性验证报告并等待用户确认。

用户可在以下三种决策中选择：
1. 补充数据源后继续
2. 授权模拟数据
3. 更换研究想法
