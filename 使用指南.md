## 2. 安装配置

### 2.1 基础安装

```bash
cd /path/to/论文-研报工作流

# 创建虚拟环境
python3 -m venv .venv && source .venv/bin/activate

# 完整安装：安装核心依赖及 extras 组中的全部常用可选依赖
# extras 包含 Tushare、akshare、yfinance、MCP、仪表盘和文档处理等集成
pip install -e ".[extras]"

# 配置 API 密钥（至少需要一个）
echo "DEEPSEEK_API_KEY=sk-your-key" > .env
```

### 2.2 扩展安装

```bash
# 计量经济扩展（DID / IV / GMM / 面板）
pip install -e ".[econometrics]"

# 向量文献库（RAG + 语义检索）
pip install -e ".[rag]"

# VLM 图表评估（深度学习）
pip install -e ".[deep-learning]"

# 云端代码执行（安全沙箱）
pip install -e ".[sandbox]"
# 需额外设置 E2B_API_KEY

# 浏览器自动化（动态网页抓取）
pip install -e ".[browser]"
playwright install chromium

# 全量安装
pip install -e ".[all]"
```

## 3. 快速开始

### 3.1 首次使用前必做

```bash
# 系统健康检查
python scripts/health_check.py --json
```

**诊断结果解读：**

|| 问题类别 | 含义 | 是否阻塞 |
|---------|------|---------|
| `network` | 网络问题 / Proxy | 部分阻塞 |
| `api_key` | API Key 缺失 | 可选功能受限 |
| `dependency` | Python 依赖缺失 | 某些模块不可用 |
| `mcp` | MCP 服务器未连接 | 数据获取受限 |
| `data_source` | 数据源不可用 | 需补充或授权模拟 |

### 3.2 最简方式

在 Cursor 对话框中直接用自然语言描述研究需求：

```
"我想研究关税政策对A股出口型企业资本结构调整速度的影响，设计一篇发表在经济研究的实证论文"
```

AI Agent 自动调用所有必要模块完成全流程。

### 3.3 脚本方式

```bash
# 完整流水线（主题 → 论文 PDF）
python scripts/agent_pipeline.py --topic "碳排放权交易对企业绿色创新的影响"

# 演示研报（使用模拟数据，无需配置）
python scripts/demo_research_report.py --stock 000001.SZ

# 分步骤执行
python scripts/research_framework/pipeline.py --mode lit-review --topic "数字金融"
```

## 4. 核心原则

1. **数据优先** — 数据验证前移到想法生成阶段（阶段 1.5），不等到阶段 5才发现无数据
2. **禁止静默 Fallback** — 模拟数据必须经用户明确授权才可使用
3. **强制交互 Checkpoint** — 每阶段完成后暂停，等待用户确认，不自动继续
4. **生成-评审分离** — 写作和 review 由不同模块处理
5. **数据溯源** — 每次数据获取记录来源和时间戳
6. **中文顶刊标准** — 经济研究 / 金融研究 / 管理世界（含完整稳健性检验）

## 5. 数据目录结构

### 5.1 外部数据放置位置

**所有外部数据文件统一放入 `data/` 目录。**

```
data/                          ← 外部数据唯一入口
├── README.md                  ← 数据目录说明
├── finance/                   ← 金融数据（CSV / Excel）
│   ├── stock_financials/      ← A股财务（利润表 / 资产负债表 / 现金流量表）
│   │   ├── annual_reports.csv
│   │   └── quarterly_reports.csv
│   ├── market_data/           ← 行情数据（日频 / 月频）
│   └── fund_data/             ← 基金数据
├── customs/                    ← 海关进出口数据（关税研究必备）
│   ├── company_import.csv      ← 企业进口明细（HS编码）
│   └── company_export.csv       ← 企业出口明细（HS编码）
├── esg/                       ← ESG 评级数据
├── macro/                     ← 宏观数据（各省 / 全国）
│   └── national_province_data_2026.xlsx
├── policy/                    ← 政策实验数据
│   └── tariff_events.csv        ← 关税政策事件（2018年中美贸易战）
├── alternative_data/          ← 另类数据
│   ├── satellite/             ← 卫星图像数据
│   ├── news_sentiment/        ← 新闻情绪数据
│   └── patent_data/           ← 专利数据
├── user_uploaded/            ← 用户临时上传（不提交 git）
└── processed/                 ← 清洗后的中间数据（自动生成）
```

### 5.2 数据提供方式

**方式一：Excel / CSV 文件**

将文件放入对应子目录，在 `FIN_BRIEF.md` 中注明：

```markdown
## 数据来源

| 数据类型 | 文件路径 | 来源 | 字段说明 |
|---------|---------|------|---------|
| A股财务 | data/finance/stock_financials/annual_reports.csv | CSMAR | 资产负债率、ROA等 |
| 海关进出口 | data/customs/company_export.csv | 海关总署 | HS编码、出口额 |
```

**方式二：MCP 自动获取**（无需手动下载）

| 数据类型 | MCP 服务器 | API Key |
|---------|---------|---------|
| A股行情 / 财务 / 融资融券 | `user-tushare` | TUSHARE_TOKEN |
| 美股 / 港股 / 期权 / ETF | `user-yfinance` | 无需 |
| 中国宏观（GDP / CPI / M2）| `user-financial` | 无需 |
| 世界银行宏观 | `user-wb-data` | 无需 |
| 学术论文 | `user-openalex` / `user-arxiv` | 无需 |

### 5.3 命名规范

| 类型 | 命名格式 | 示例 |
|------|---------|------|
| 面板数据 | `{topic}_{frequency}_{year_range}.csv` | `a_share_annual_2010_2024.csv` |
| 事件数据 | `{event_type}_{date}.csv` | `tariff_2018_07.csv` |
| 评级数据 | `{provider}_{rating_type}_{date}.csv` | `msci_esg_2024.csv` |
| 宏观数据 | `{region}_{indicator}_{year_range}.xlsx` | `china_province_gdp_2010_2023.xlsx` |

### 5.4 禁止事项

- ❌ 不要将数据文件放入 `scripts/` 或 `mcp_servers/` 目录
- ❌ 不要在 `output/` 目录存放原始数据（output 是输出目录）
- ❌ 不要提交超过 10MB 的数据文件到 git（使用 `.gitignore` 或 Git LFS）
- ❌ 敏感数据（包含股票账户、身份证等）禁止放入项目目录

### 9.2 章节写作顺序（推荐）

```
1. Data & Methodology — 数据和方法最清晰，先写
2. Main Results — 实证结果
3. Introduction — 读者还不知道背景
4. Literature Review — 边写边补充文献
5. Mechanism / Heterogeneity — 延伸分析
6. Conclusion — 最后总结
7. Abstract — 全文完成后写
```

### 9.3 写作标准

| 要素 | 要求 |
|------|------|
| 引用 | 每段至少1个引用（除摘要） |
| 假说推导 | 理论 → 机制 → 假说（H1 / H2 / H3）|
| 统计量 | 系数、SE、t值、p值、样本量、R² |
| 经济显著性 | 系数的经济含义（元 / % / 标准差单位）|

## 13. 常见问题

### Q1: 启动时报错 `ModuleNotFoundError`

```bash
source .venv/bin/activate
pip install -e ".[extras]"
```

### Q2: 没有 TUSHARE_TOKEN 还能做 A股研究吗？

可以。有三个选项：
1. 使用 `user-financial`（akshare）的中国宏观数据（无需 Key，但个股财务有限）
2. 手动下载 Excel 放入 `data/finance/` 目录
3. 使用 `user-eastmoney-reports` 获取研报中的数据（无需 Key）

### Q3: 关税研究需要哪些数据？

至少需要：
- A股上市公司财务数据（Tushare / CSMAR）
- 海关进出口数据（CSMAR，含 HS 编码）
- 关税政策事件（手动整理放入 `data/policy/`）

### Q4: 可以只做文献综述或只跑实证吗？

可以。每个 Skill 都是独立的：

```bash
python scripts/research_framework/pipeline.py --mode lit-review --topic "..."
python scripts/research_framework/pipeline.py --mode regression --topic "..."
```

### Q7: 计量回归结果不显著

1. 检查数据质量：`python scripts/research_framework/data_validator.py`
2. 尝试不同控制变量组合

4. 检查是否存在样本选择偏差
