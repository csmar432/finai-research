# AI Use Disclosure Templates for Academic Papers

> When using AI-assisted tools in academic writing, disclosure is required by most journals.

## English Templates

### General Disclosure
```markdown
**AI Assistance Statement.** This paper was developed with the assistance of
large language models (LLMs) for: literature review, data analysis,
statistical programming, and manuscript drafting. All AI-generated content
was reviewed, validated, and revised by the authors. The authors take
full responsibility for the accuracy and originality of all content.
```

### JF / RFS Standard
```markdown
**AI Assistance.** We used [tool] to assist with [specific tasks].
All analyses and conclusions were independently verified by the authors.
```

## Chinese Templates

### 通用 AI 辅助披露
```markdown
**AI 辅助说明。** 本研究使用大语言模型（LLM）辅助文献检索、数据分析和论文撰写。
所有 AI 生成内容均经作者审核验证，作者对所有内容的准确性和原创性负责。
```

### 经济研究 / 金融研究 / 管理世界
```markdown
**AI 辅助说明。** 本研究使用 AI 辅助工具（[工具名称]）辅助文献综述、实证分析和论文撰写。
所有 AI 生成内容经作者审核验证，符合期刊关于 AI 使用披露的政策要求。
```

## Best Practices
1. Always disclose — name the tool and describe the task
2. Own the content — make clear you reviewed everything
3. Check journal policy — some journals require specific LLM name disclosure
4. Preserve transparency — keep logs of AI interactions
