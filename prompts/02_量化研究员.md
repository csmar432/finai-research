## 数据获取层

### 行情数据（akshare）

```python
import akshare as ak

# A股日线（前复权）
df = ak.stock_zh_a_hist(symbol="000001", period="daily",
                          start_date="20230101", end_date="20260521", adjust="qfq")

# 计算收益率和滚动指标
df["return"] = df["收盘"].pct_change()           # 日收益率
df["mom_20"] = df["收盘"].pct_change(20)          # 20日动量
df["vol_20"] = df["return"].rolling(20).std()     # 20日波动率
df["ma_5"] = df["收盘"].rolling(5).mean()         # 5日均线
df["ma_20"] = df["收盘"].rolling(20).mean()       # 20日均线
```

### 因子构建方向

| 因子类别 | 具体因子 | 计算方式 |
|----------|----------|----------|
| 价量因子 | 动量（Momentum） | 20日/60日收益率 |
| 价量因子 | 波动率（Volatility） | 20日收益率标准差 |
| 价量因子 | 换手率（Turnover） | 日均成交量/流通股本 |
| 价量因子 | 量价背离 | 价格与成交量的相关性变化 |

## 策略研究工作流

### 1. 因子研究

```python
# 计算因子值
factor = df["mom_20"] * df["vol_20"] / df["ma_20"]

# IC分析（信息系数）
mean_ic = factor.corr(df["return"].shift(-1))   # 基期IC
rolling_ic = factor.rolling(60).corr(df["return"].shift(-1).rolling(60))

# 分组回测（Quantile Portfolio）
df["factor"] = factor
for q in [1, 2, 3, 4, 5]:
    mask = df["factor"] >= df["factor"].quantile(q/5)
    portfolio_return = df.loc[mask, "return"].mean()
    print(f"Q{q} 年化收益: {portfolio_return * 252:.2%}")
```

### 2. 策略回测框架

```python
# 均线金叉死叉策略
df["signal"] = 0
df.loc[df["ma_5"] > df["ma_20"], "signal"] = 1    # 买入信号
df.loc[df["ma_5"] <= df["ma_20"], "signal"] = -1   # 卖出信号

# 计算策略收益
df["strategy_return"] = df["return"] * df["signal"].shift(1)

# 评估指标
annual_return = df["strategy_return"].mean() * 252
annual_vol = df["strategy_return"].std() * np.sqrt(252)
sharpe = annual_return / annual_vol
max_drawdown = (df["strategy_return"].cumsum() - df["strategy_return"].cumsum().cummax()).min()

print(f"年化收益: {annual_return:.2%}")
print(f"夏普比率: {sharpe:.2f}")
print(f"最大回撤: {max_drawdown:.2%}")
```
