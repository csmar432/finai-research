# 【模板7】论文选题与大纲生成

## 工作流说明

此模板驱动 DeepSeek 自动完成论文选题与大纲设计：
1. 研究问题定位（将模糊方向聚焦为可操作问题）
2. 论文大纲生成（各章节内容概要）
3. 创新点梳理（3-5 条）
4. 研究路线图（时间规划）

---

## 使用方法

**直接复制以下模板，发给 Cursor Agent：**

---

```
请帮我设计一篇论文的选题与大纲：

## 研究方向
[填写你的研究方向，例如：深度学习 量化交易]

## 目标期刊/会议（可选）
[例如：NeurIPS, JFE, ACL, ICML]

## 请输出
1. 候选论文标题（3 个备选）
2. 研究问题定位（1-2 个具体可操作的问题）
3. 主要创新点（3-5 条，说明创新类型和具体内容）
4. 论文大纲（每个章节的核心内容、关键问题、逻辑衔接）
5. 研究路线图（数据准备→方法设计→实验验证→论文写作）
6. 预期结果
```

---

## 脚本命令

```bash
python scripts/paper_outline.py "[研究方向]" --venue "[目标期刊]" --save
python scripts/paper_outline.py "深度学习 量化交易" --venue "NeurIPS" --save
```
