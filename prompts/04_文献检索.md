# 【模板4】文献检索与综述生成

## 工作流说明

此模板驱动一个完整的四步文献研究流程：

```
Step 1  DeepSeek 检索文献线索
        ↓
Step 2  自动下载论文原文（arXiv PDF）
        ↓
Step 3  DeepSeek 深度解析（结构化摘要）
        ↓
Step 4  DeepSeek 生成文献综述
```

---

## 使用方法

**直接复制以下模板，填入参数，发给 Cursor Agent：**

---

```
请帮我完成以下文献研究与综述任务：

## 研究主题
[填写你的研究方向，例如：深度学习 量化交易]

## 检索参数
- 论文数量：推荐 [5-10] 篇
- 时间范围：近 [3-5] 年
- 优先来源：[arXiv / NeurIPS / ICML / KDD / AAAI / JFE / JFQA 等]

## 结构化分析要求
请对每篇论文提取以下信息：
1. 研究问题：该论文要解决什么问题？
2. 自变量/输入特征：使用了哪些输入变量？
3. 因变量/评价指标：如收益率、准确率、夏普比率等
4. 模型/方法：使用的核心模型（LSTM、Transformer、强化学习等）
5. 数据来源与样本期间：如 A股日线、Wind 数据、2015-2020 年
6. 主要结论：3-5 条
7. 局限性：论文的不足之处

## 文献综述要求
基于上述论文，生成一份文献综述，包含：
1. 研究方法对比（各论文方法有何异同）
2. 核心发现总结（共识与分歧）
3. 变量体系梳理（该领域常用变量）
4. 研究局限性（共同不足）
5. 未来研究方向
6. 对我研究的启发

## 输出格式
1. 结构化论文列表（Markdown 表格）
2. 完整文献综述（Markdown 格式，800-1500 字）
3. 将综述保存到 knowledge/reviews/ 目录下

## 附：脚本命令（如需手动执行）
# 完整流程（检索 + 下载 + 解析 + 综述）
python scripts/literature_search.py "[研究主题]" --topic "[主题名]" -n [数量]

# 跳过下载（只解析摘要信息，速度更快）
python scripts/literature_search.py "[研究主题]" --skip-download

# 不生成综述
python scripts/literature_search.py "[研究主题]" --no-review
```

## 示例

```
请帮我完成以下文献研究与综述任务：

研究主题：强化学习 做市商算法
检索参数：推荐 8 篇，近 5 年，优先 NeurIPS、ICML、JFE
结构化分析要求：提取变量、模型、数据来源、结论
文献综述要求：方法对比、核心发现、局限性、未来方向
```

---

## 可调参数说明

| 参数 | 位置 | 说明 |
|------|------|------|
| 研究主题 | 模板第一行 | 核心检索关键词 |
| 论文数量 | `--topic` 后的 `-n` | 默认 10 篇，建议 5-8 篇 |
| 时间范围 | prompt 中指定 | 近 3-5 年 |
| 优先来源 | prompt 中指定 | 顶会/核心期刊 |
| 跳过下载 | `--skip-download` | 无 arXiv ID 时使用 |
| 跳过综述 | `--no-review` | 仅检索和分析，不生成综述 |

---

## 生成的文件

| 文件 | 位置 |
|------|------|
| 文献综述 | `knowledge/reviews/[主题]_[时间戳].md` |
| 论文原文 | `knowledge/papers_fulltext/[arXiv ID].pdf` |
| 论文文字 | `knowledge/papers_fulltext/[arXiv ID].txt` |
| 论文元信息 | `knowledge/papers_meta/[arXiv ID].json` |
| 检索记录 | `knowledge/papers/raw_search_[主题]_[时间戳].txt` |
