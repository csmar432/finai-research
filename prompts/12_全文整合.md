# 【模板12】全文整合

## 工作流说明

将所有章节草稿拼接成完整论文草稿，并生成 Abstract。

---

## 使用方法

---

```
请帮我整合完整论文：

## 论文标题
[论文标题]

## 研究主题
[你的研究方向]

## 请整合
1. 读取 knowledge/outlines/ 下的最新大纲
2. 读取 knowledge/chapters/ 下的所有章节草稿
3. 读取 knowledge/reviews/ 下的最新文献综述
4. 生成完整论文，包含：
   - Title
   - Abstract（约 200 字）
   - Introduction
   - Related Work
   - Methodology
   - Experiments
   - Conclusion
5. 补充缺失章节，衔接各部分
6. 保存到 output/ 目录
```

---

## 脚本命令

```bash
# 完整整合（使用最新材料）
python scripts/paper_assembler.py

# 指定标题
python scripts/paper_assembler.py --title "论文标题"

# 指定章节文件
python scripts/paper_assembler.py --source knowledge/chapters/xxx.md

# 输出 LaTeX 格式
python scripts/paper_assembler.py --format latex

# 输出 Word 格式
python scripts/paper_assembler.py --format word
```

---

## 生成的文件

| 文件 | 位置 |
|------|------|
| 完整论文草稿 | `output/[主题]_[时间戳].md` |
| LaTeX 文件 | `output/[主题]_[时间戳].tex`（需 pandoc） |
