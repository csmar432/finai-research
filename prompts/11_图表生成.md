# 【模板11】论文图表生成

## 工作流说明

DeepSeek 为论文生成专业图表：
1. **模型架构图**（Graphviz DOT 代码）
2. **实验结果可视化**（Python/matplotlib 代码）
3. **LaTeX 表格**（可直接编译）

---

## 使用方法

---

```
请帮我生成论文图表：

## 图表类型（可多选）
[架构图 / 实验结果图 / 表格]

## 研究主题/方法
[你的研究方向]

## 方法描述（用于架构图）
[模型的核心组件和数据流向]

## 实验结果数据（用于图表）
[表格数据或实验描述]

## 请生成

### 架构图
- Graphviz DOT 代码（可直接 dot 命令编译）
- 说明各模块的作用

### 实验结果可视化
- Python matplotlib/seaborn 代码（300 DPI，适合论文）
- 图表说明

### LaTeX 表格
- 主实验对比表（booktabs 风格）
- 消融实验表
- 数据集描述表
```

---

## 脚本命令

```bash
# 生成全部图表
python scripts/paper_visualizer.py --topic "[研究主题]" --save

# 仅生成架构图
python scripts/paper_visualizer.py --type architecture --topic "[研究主题]"

# 生成实验可视化
python scripts/paper_visualizer.py --type experiment --results "[实验数据]"

# 生成 LaTeX 表格
python scripts/paper_visualizer.py --type table --data "[表格数据]"
```

### 编译图表命令

```bash
# 架构图（需安装 Graphviz）
dot -Tpdf architecture.dot -o architecture.pdf
dot -Tsvg architecture.dot -o architecture.svg

# 实验可视化
python generate_plot.py

# LaTeX 表格（直接复制到 .tex 文件）
pdflatex paper.tex
```
