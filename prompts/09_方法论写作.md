# 【模板9】Methodology 方法论写作

## 工作流说明

根据论文大纲和实验设计，DeepSeek 生成：
1. **Methodology（方法论）** — 问题定义、模型/算法设计、训练策略、复杂度分析

---

## 使用方法

---

```
请帮我撰写 Methodology（方法论）章节：

## 研究主题
[你的研究方向]

## 论文大纲（方法论部分）
[从 paper_outline.py 生成的大纲中提取方法论描述]

## 实验设计说明（可选）
[你计划做的实验，如：数据集、基线方法、评估指标]

## 请生成
请按以下结构撰写（目标 2500 字）：

### 3.1 问题定义
- 形式化定义输入、输出
- 用数学符号说明研究问题
- 定义评估指标

### 3.2 预备知识（可选）
- 核心技术背景（2-3 段）

### 3.3 模型/算法设计
- 详细描述提出的模型架构或算法
- 包含关键组件说明
- 伪代码或流程图

### 3.4 训练策略
- 损失函数、优化器、学习率调度
- 正则化、数据增强

### 3.5 复杂度分析（可选）
- 时间/空间复杂度

## 风格要求
- 使用 LaTeX 格式的数学公式
- 每个决策都有动机说明
- 主动语态（We propose / We design）
```

---

## 脚本命令

```bash
python scripts/paper_method_writer.py
python scripts/paper_method_writer.py knowledge/outlines/xxx.md --save
python scripts/paper_method_writer.py --experiment "实验描述"
```
