# 【模板5】研报框架

## 如何使用
复制后填入公司名，发给 Cursor Agent。

---

帮我写一份 [公司/行业名] 的研究报告：

**基本信息**：
- 分析对象：[公司/行业]
- 覆盖期：[2021-2025]
- 评级：[推荐/增持/中性/减持]
- 目标价：[XX元]

**数据来源**：
- 财务数据：公司年报、季报
- 行业数据：[行业协会 / 公司公告 / Wind]
- 宏观数据：[国家统计局 / 央行]

**分析重点**：
1. [重点1，例如：新能源电池竞争格局]
2. [重点2，例如：海外扩张风险]
3. [重点3]

**现在开始**，按以下结构输出研报：

```
# [公司名] 深度研究报告

## 执行摘要

## 一、行业概览

## 二、公司基本面

## 三、财务分析

## 四、估值分析

## 五、投资建议

## 风险提示
```
