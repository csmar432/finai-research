# 【模板8】论文引言与相关工作写作

## 工作流说明

根据论文大纲和文献综述，DeepSeek 生成：
1. **Introduction（引言）** — 背景、问题、挑战、方法概述、贡献、论文结构
2. **Related Work（相关工作）** — 技术基础、方法分类、数据集与基准、与本文关联

---

## 使用方法

---

```
请帮我撰写论文章节：

## 研究主题
[你的研究方向]

## 大纲文件
knowledge/outlines/ 下的最新大纲，或直接提供：

创新点：
- [创新点1]
- [创新点2]
- [创新点3]

## 请生成
1. Introduction（约 800 字）
   - 领域背景（1-2 段）
   - 研究问题与挑战（1-2 段）
   - 本文方法概述（1 段）
   - 主要贡献（条目形式，We propose...）
   - 论文结构（1 段）

2. Related Work（约 1200 字）
   - 技术基础（2-3 段）
   - 现有方法分类（按技术路线分组，每组分析优缺点）
   - 数据集与基准（1 段）
   - 与本文的关联（1 段）

## 参考文献
请在 Related Work 中引用 knowledge/papers/index.json 中的论文，用 [1] [2] 格式
```

---

## 脚本命令

```bash
# 生成全部（Introduction + Related Work）
python scripts/paper_intro_writer.py

# 仅生成 Introduction
python scripts/paper_intro_writer.py --intro-only

# 指定大纲文件
python scripts/paper_intro_writer.py knowledge/outlines/xxx.md --save

# 指定研究主题
python scripts/paper_intro_writer.py --topic "深度学习 量化交易"
```
