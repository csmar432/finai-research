# 【模板10】实验与结论写作

## 工作流说明

根据实验结果和论文大纲，DeepSeek 生成：
1. **Experiments（实验）** — 数据集、基线、设置、结果、消融、敏感性分析
2. **Conclusion（结论）** — 总结、贡献回顾、局限性、未来工作

---

## 使用方法

---

```
请帮我撰写实验与结论章节：

## 研究主题
[你的研究方向]

## 实验结果数据
[描述你的实验结果，或留空让 AI 生成参考模板]

## 请生成

### Experiment（约 2000 字）
1. 数据集与基线方法（来源、规模、时间范围、划分）
2. 实验设置（硬件、软件、超参数、评估指标）
3. 主实验结果（表格形式，与基线对比，显著性检验）
4. 消融实验（各组件贡献）
5. 敏感性分析（超参数、数据规模）

### Conclusion（约 400 字）
1. 总结（研究问题、提出方法、主要结果）
2. 局限性（数据集、计算成本、未覆盖场景）
3. 未来工作（2-3 个具体可操作的改进方向）

## 格式要求
- 使用 LaTeX 表格格式
- 标注显著性：∗p<0.05, ∗∗p<0.01, ∗∗∗p<0.001
```

---

## 脚本命令

```bash
# 生成全部（Experiment + Conclusion）
python scripts/paper_experiment_writer.py

# 仅生成 Experiment
python scripts/paper_experiment_writer.py --chapter experiment

# 仅生成 Conclusion
python scripts/paper_experiment_writer.py --chapter conclusion

# 指定实验结果
python scripts/paper_experiment_writer.py --results "你的实验结果描述" --save
```
