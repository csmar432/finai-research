# 【模板13】论文润色与查重

## 工作流说明

对论文草稿进行：
1. **英文润色**（Grammar/Style/Tone）
2. **中文润色**（语言精炼、学术化）
3. **简单查重**（模板句式检测）
4. **多轮迭代优化**

---

## 使用方法

---

```
请帮我润色论文：

## 论文文件
[你的论文文件路径，如：output/xxx.md]

## 论文语言
[english / chinese]

## 润色级别
- light（轻微）：仅修正语法错误
- standard（标准）：语法+句子结构+可读性
- intensive（深度）：全面优化学术风格

## 查重
[是否需要查重检测]

## 请执行
1. 润色论文（指定语言和级别）
2. 查重检测（如果需要）
3. 生成润色反馈报告
4. 保存润色后的全文
```

---

## 脚本命令

```bash
# 中文润色（标准）
python scripts/paper_polisher.py output/xxx.md --lang chinese

# 英文润色（深度）
python scripts/paper_polisher.py output/xxx.md --lang english --level intensive

# 查重检测
python scripts/paper_polisher.py output/xxx.md --check-plagiarism

# 多轮迭代润色
python scripts/paper_polisher.py output/xxx.md --iterate 2 --lang chinese

# 完整润色 + 查重
python scripts/paper_polisher.py output/xxx.md --lang chinese --check-plagiarism --iterate 2 --save
```

## 注意事项

- AI 润色仅为辅助，**实际重复率需使用 iThenticate / Turnitin 等专业工具**
- 润色后请仔细核对技术内容是否准确
- 建议保留原文备份，以便对比
